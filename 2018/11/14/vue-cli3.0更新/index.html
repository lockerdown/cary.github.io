    可视化安装工具

    安装

    yarn global add @/vue-cli
    vue create <project-name>
    也可以通过vue ui来启动可视化安装工具

    注意点

    如果一开始安装了vue-cli需要先卸载之前的版本
    yarn global remove vue-cli
    查询版本:
    vue --version
