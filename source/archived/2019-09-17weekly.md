### 本周技术难点及收获

``` javascript
    const flatArray = arr => [].concat(...arr);
```
主要作用就是把多维数组拍平变成一维数组.
应用场景就是请求多个渠道的红包列表时只需要循环一次就行,而不是循环多次.
